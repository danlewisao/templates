# Assets

Store any external assets and files referenced by your architecture documentation in subfolders under this folder e.g
images, plant uml files, PDFs, Word etc.

```
assets
|__ png                 
|   |__ diagram1.png
|   |__ diagram2.png
|__ puml                                              
|   |__ context.puml
|   |__ container.puml
|__ pdf                                              
    |__ vendor-docs.pdf
```
